[ 인텔리제이 깃허브 연동 ]
    1. 인텔리제이 상단메뉴 -> [ 파일 ] -> [설정]
    2. 사이드메뉴 [ 버전관리 ] -> [Github]
    3. [계정추가] ---> guthub통해 --> 웹 깃허브 로그인후 인증

[ 인텔리제이 깃허브 커밋계정 정보 ]
    1. 인텔리제이 왼쪽 하단 메뉴 [ 터미널 : alt+f12 ]
    2. 터미널 상단 [v] 아이콘 클릭 --> [ git bash ]
    3. git config --global user.name
    git config --global user.email
